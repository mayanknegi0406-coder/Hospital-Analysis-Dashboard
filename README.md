# 🏥 Hospital Dashboard (Power BI)

This project presents an interactive **Hospital Dashboard** built using Power BI to analyze patient data, hospital performance, and operational insights.

## 🔍 Key Insights
- Patient admission and discharge trends
- Department-wise patient distribution
- Bed occupancy and hospital utilization
- Treatment and service performance analysis
- Overall hospital efficiency indicators

## 📈 Features
- Patient and admission analysis
- Department-wise performance tracking
- Bed occupancy visualization
- Time-based trends and insights
- Interactive filters for dynamic reporting

## 🛠 Tools Used
- Power BI  
- Data Analysis  
- Data Visualization  

## 🎯 Purpose
The objective of this project is to provide insights into **hospital operations, patient trends, and resource utilization** to support better decision-making and showcase data analytics skills.
